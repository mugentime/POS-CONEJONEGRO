BÚSQUEDA DE CLIENTES

Sistema de autocompletado inteligente para selección rápida de clientes

Volver a Gestión de Clientes

Demostración Interactiva

Escribe en el campo de búsqueda para ver el autocompletado en acción. El sistema busca por nombre, teléfono o email y muestra resultados en tiempo real.

Cliente Seleccionado

Instrucciones de Uso:

  • Escribir: Comienza a escribir cualquier parte del nombre, teléfono o email
  • Navegar: Usa las flechas ↑ ↓ para navegar por los resultados
  • Seleccionar: Presiona Enter o haz clic en un resultado
  • Crear: Si no existe el cliente, aparecerá la opción de crearlo
  • Cancelar: Presiona Escape para cerrar el dropdown

Implementación

Para usar este componente en cualquier página, simplemente incluye el script y inicializa el componente en cualquier input: 

// HTML
<input type="text" id="customer-input" />

// JavaScript
const searchInput = document.getElementById('customer-input');
const customerSearch = new CustomerSearch(searchInput, {
    onSelect: (customer) => {
        console.log('Cliente seleccionado:', customer);
    },
    onCreate: (name) => {
        console.log('Crear cliente:', name);
    }
});

// Obtener cliente seleccionado
const selectedCustomer = customerSearch.getSelectedCustomer();

API Endpoint

El componente utiliza el endpoint /api/customers/search/{query} que devuelve resultados en formato JSON: 

GET /api/customers/search/maria
